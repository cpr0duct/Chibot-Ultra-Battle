# ChUB 2000 Web Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Recreate Chibot Ultra Battle as a web application hosted on Raspberry Pi with faithful game mechanics, original file format compatibility, web multiplayer, full editor suite, and Jukebox audio.

**Architecture:** Monolithic Node.js server (Express + Socket.IO) on port 9012. Shared game engine modules between server and client. Vanilla HTML/CSS/JS frontend. Original VB6 data files (.CH2, .AN2, .ITM, .W2K, .INI) parsed natively.

**Tech Stack:** Node.js 22, Express, Socket.IO, Vitest (testing), Web Audio API, MIDI.js

**Reference:** Design spec at `docs/superpowers/specs/2026-03-28-chub-2000-web-design.md`. Original VB6 source at `ChiSource/` and `ChiSource_fetched/`.

// parsers/string-vars.js
// String variable substitution engine for ChUB data files and battle messages

export function substituteVars(template, vars = {}) {
  if (!template) return '';
  return template
    .replace(/%SN/g, vars.SN ?? '')
    .replace(/%S2/g, vars.S2 ?? '')
    .replace(/%T/g, vars.T ?? '')
    .replace(/%Y/g, vars.Y ?? '');
}

export function stripQuotes(str) {
  if (!str) return '';
  const trimmed = str.trim();
  if (trimmed.startsWith('"') && trimmed.endsWith('"')) {
    return trimmed.slice(1, -1);
  }
  return trimmed;
}

export function parseQuotedString(line) {
  return stripQuotes(line.trim());
}
